# Activities Overview


Moodle includes resources and activities.

- A **resource** usually provides content for learners to read, watch, or download.
- An **activity** usually requires learner interaction, submission, participation, or assessment.

Common resources:

| Resource | Use Case |
|---|---|
| File | Upload a PDF, Word document, spreadsheet, or presentation |
| Folder | Share multiple files in one place |
| Page | Create simple web content inside Moodle |
| URL | Link to an external website or resource |
| Book | Create structured multi-page learning content |
| Label/Text and media area | Add instructions, headings, or embedded media |

Common activities:

| Activity | Use Case |
|---|---|
| Assignment | Students submit files or online text for grading |
| Quiz | Online tests, quizzes, exams, or practice checks |
| Forum | Discussion and announcements |
| Choice | Simple polling or selection activity |
| Feedback | Collect survey-style responses |
| Glossary | Shared list of terms and definitions |
| Lesson | Branching learning path with pages and questions |
| Workshop | Peer assessment activity |
| H5P | Interactive content such as presentations and quizzes |

Administrators do not always create learning content, but they should understand these tools because support requests often involve activity settings, access issues, grades, or file problems.
