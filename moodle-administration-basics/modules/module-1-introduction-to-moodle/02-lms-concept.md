# LMS Concept


An LMS, or Learning Management System, is software used to manage learning activities.

Common LMS functions include:

- Course creation and organization.
- User enrolment and access control.
- Uploading files and learning materials.
- Online quizzes and assignments.
- Discussion forums and communication tools.
- Gradebook management.
- Course completion tracking.
- Reporting and analytics.

For a technical support team, it is important to understand that the LMS is used by different user groups with different expectations:

| User Group | Typical Needs |
|---|---|
| Students | Access courses, submit assignments, take quizzes, view grades |
| Teachers | Upload materials, create activities, grade work, communicate with students |
| Administrators | Configure the site, manage users, manage courses, monitor system status |
| IT Support | Resolve login issues, investigate errors, maintain availability and performance |
| Management | View reports, monitor usage, support academic processes |
