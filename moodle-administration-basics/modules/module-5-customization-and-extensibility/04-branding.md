# Branding


Branding makes Moodle feel like part of the organization’s digital environment.

Branding elements include:

- Site name.
- Logo.
- Favicon.
- Primary colors.
- Footer text.
- Support contact information.
- Login instructions.
- Dashboard messaging.
- Front page announcements.

Good branding supports users by making the site recognizable and trustworthy. However, branding should not reduce usability. For example, low-contrast colors may look attractive but make text hard to read.
