# Common Issues


Common Moodle support issues include:

| Issue | Possible Causes |
|---|---|
| User cannot log in | Wrong password, suspended account, SSO issue, authentication plugin problem |
| User cannot see a course | Not enrolled, course hidden, wrong role, availability restriction |
| Teacher cannot edit course | Missing teacher role, editing disabled, category permissions issue |
| File upload fails | File too large, PHP upload limit, disk full, permission issue |
| Emails not sent | Cron not running, SMTP misconfiguration, mail server problem |
| Quiz is slow | High concurrent users, database load, question behavior, server capacity |
| Grades not visible | Gradebook settings, hidden grade item, release timing, permissions |
| Theme looks broken | Cache issue, theme setting error, custom CSS, plugin conflict |
| Plugin error after upgrade | Incompatible plugin, missing dependency, failed upgrade step |
| Backup fails | Timeout, storage issue, large course, permissions, cron issue |
